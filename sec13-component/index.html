コンポーネント

グローバル登録

vue-componentメソッドによるコンポーネント定義はグローバル登録と呼ばれ、全てのvueインスタンスから利用できる。コンポーネント名をview側にタグとして記載し使用

定義はインスタンスが作成される前に行う

ローカル登録

jsファイルの変数にtemplateを代入し、vueインスタンス内でcomponentsを定義することで、ローカル登録ができる。view使用方法はグローバルと同じく、タグとして利用

定義はインスタンスが作成される前に行う

コンポーネント名はハイフンを一つ以上含むケバブケースにする

  • hello-component・・・OK
  • button-counter・・・OK
  • hello・・・NG
  • helloComponent・・・NG

オプション

それぞれが独自にデータを保持する。コンポーネントを作成する度に、新しいインスタンスが作成される。

コンポーネントは再利用可能なvueインスタンス。methodsやデータ、コンピューテッド、ウオッチなどのnew vueする場合と同じオプションを利用可能。

例外はelのようなルート固有のオプション

注意

1.コンポーネントにおけるdataの指定方法について、関数である必要がある。直接、オブジェクトとして定義できない。

2.テンプレートの要素は単一である必要がある。

×　template: 

'<span>count: </span><button v-on:click="count++">{ { count }}</button>'

二つ以上使用する場合、divタグで囲う必要がある

○　template: 

'<div><span>count: </span><button v-on:click="count++">{ { count }}</button></div>'
